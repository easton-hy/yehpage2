Publish ME5001 Proj as .exe File

The only input of my ME5001 project is the IP address, so I used PyQt5 to design a UI to text the IP address and publish the UI as a exe file. At first, I used pyinstaller and auto-py-to-exe (also based on pyinstaller) to publish the python file, but because of torch the exe file cannot be used. The pyinstaller has limitation in publishing a complicate project with the packages like torch, tensorflow. Instead, I was trying to use nuitka. Here is the comparison of nuitka and pyinstaller. In a word, pyinstaller is suitable for small python project with advantages of easy getting start, while the nuitka suits for complex and large project. The nuitka's parameters are list here. I hadn't publish project by nuitka yet, and I spent five hours for the first trial to publish the project.
